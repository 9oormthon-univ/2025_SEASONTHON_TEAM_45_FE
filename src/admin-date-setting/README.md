# 관리자 날짜, 시간 관리 페이지 컴포넌트 구조

## 1. AdminSetting
- **설명:** 날짜, 시간 관리 메인 화면
- **역할:** 전체 목록과 페이지 레이아웃 관리

## 2. AdminHeader
### component 파일로 이동
- **설명:** 관리자 헤더
- **역할:** 상단 헤더, 병원 로고, 관리자 정보 등 담당

## 3. TimeSlots
- **설명:** 환자 상태 대기열
- **역할:** 각 상태별 대기 목록 컬럼 표시
- **기능 추가 (예정):** 환자 호출시 창구 선택 가능

## 4. UserCardRow
- **설명:** 환자 정보 카드 
- **역할:** 개별 환자 정보 요약 카드 - 가로 정렬 버전

## 5. UserModal
### component 파일로 이동
- **설명:** 환자 정보 모달
- **역할:** 환자 상세 정보 팝업 모달
- **기능 추가 (예정):** 디자인 적용 예정

## 6. UserAdminList
- **설명:** 해당 날짜 예약 환자 정보
- **역할:** 선택한 날짜의 예약 환자 열람

## 7. UserMenu
### component 파일로 이동
- **설명:** 관리자 계정 드롭 다운
- **역할:** 로그아웃 / 관리자 페이지 이동 기능
- **기능 추가 (예정):** 디자인 적용 예정
